# Search-Engine-Information-Retrieval-System
A search engine developed over the Project Gutenberg dataset containing more than 70,000 text documents. The project implemented indexing, ranking, and retrieval techniques including TF-IDF and BM25 to efficiently retrieve and rank relevant documents based on user queries.

Technologies
1. Python
2. Information Retrieval
3. TF-IDF
4. BM25
5. Data Processing

Key Features
1. Document Indexing
2. Query Processing
3. Document Ranking
4. Metadata-Based Search
5. Large-Scale Text Retrieval
