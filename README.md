# Search Engine / Information Retrieval System

## Overview

A search engine developed over the Project Gutenberg dataset containing more than 70,000 text documents. The project focused on implementing core information retrieval techniques to efficiently index, retrieve, and rank relevant documents based on user search queries. It explored traditional search algorithms and ranking models commonly used in modern search engines.

## Technologies Used

- Python
- BM25
- TF-IDF
- Information Retrieval
- Pandas

## Key Features

- Built an inverted index to support efficient document retrieval.
- Implemented TF-IDF and BM25 ranking algorithms to improve search relevance.
- Developed query processing and document ranking pipelines for fast and accurate search results.
- Enabled metadata-based search using document attributes such as title, author, language, and bookshelf categories.
- Generated ranked search results based on user queries across a large-scale text collection.

## Dataset

- Project Gutenberg Dataset
- 70,000+ public domain text documents
- Metadata including title, author, language, and bookshelf categories

## Results

Successfully developed a functional search engine capable of retrieving and ranking relevant documents using established information retrieval techniques. The project demonstrated how indexing and ranking algorithms improve search accuracy and retrieval efficiency across large document collections.

## Future Improvements

- Implement semantic search using transformer-based embeddings.
- Add query auto-completion and spelling correction.
- Develop a web-based search interface.
- Integrate hybrid search combining keyword and vector retrieval.
